## 🛠 Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- C++
- Data Structures (Queue - FIFO)

## ✨ Features

- Parking Slot Allocation
- Vehicle Entry & Exit
- Waiting Queue (FIFO)
- Parking Fee Calculation
- Dashboard Analytics
- Parking History
- C++ Slot Allocation Algorithm

## 📦 Installation


### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
